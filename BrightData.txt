Monitor Progress API
Use this API to check your data collection status. Enter the snapshot id provided by the "Trigger Data Collection API" response. It will return "running" during data gathering and "ready" when the data is available.
curl -H "Authorization: Bearer 098f7911c78fc644d0edd08ada631f41dcfa3d9ef63e44a224d9105720279a1f" "https://api.brightdata.com/datasets/v3/progress/"
